# 構造物系100件拡張メモ

辞書を100項目へ拡張するため、構造物・都市・建築空間から逆引きしやすい項目を55件追加しました。

追加項目は「○○パンク」に無理に寄せず、実際の逆引きで使いやすい建築・都市タイプ名を優先しています。

## 追加した方向性

- 地形と一体化した住居: 崖住居、洞窟都市、段丘都市
- 防御都市: 城塞都市、城壁都市、シタデル建築
- 水辺と砂漠の都市: 高床建築、運河都市、オアシス都市、港湾工業地帯
- 研究・インフラ施設: 極地基地建築、研究ステーション建築、データセンター建築、発電所建築
- 宇宙・閉鎖環境: 軌道居住区、世代宇宙船内部、ドーム都市、バイオドーム
- 交通建築: 空港都市、交通結節点建築、モノレール都市、高架橋建築
- 公共・知識空間: 図書館建築、アーカイブ建築、美術館建築、市場ホール建築
- 宗教・儀式空間: 修道院建築、回廊建築、ジッグラト建築、ピラミッド都市

## 代表画像について

新規55件は `representative_image` を空の状態で追加しています。

アプリでは `代表画像未設定` タブに表示されます。代表画像を追加する場合は、1ジャンル1枚を基本に、出典・作者・ライセンスを保存してください。
